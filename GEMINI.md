# Project: Personal Website

## Overview

Your task is to create my personal website, bsalbilla06.com.

## General Instructions

- This is a static website built with Tailwind CSS.
- App deployment is on GitHub Pages.
- The website is composed of standalone HTML files (index.html, about.html, experience.html, connect.html).
- Assets are located in the `/static` directory.

## Style

- The website should be lightweight and fun, while still fully functional
- The background of the website should look like graph paper
- Images and text should look handwritten by pencil
- During initial development, use simple shapes or generic placeholders for images. Once the layout is finalized, provide me with the prompts to generate the final handwritten pencil images.
- Cards should look like paper taped onto the page
- The navbar should follow the aforementioned styles, the active page should be circled with what looks like "red crayon"

# Content

- Include the following pages: Landing/Home page, About Me page, Experience page, Contact page
- Use placeholder lorem ipsum text where my input is necessary
